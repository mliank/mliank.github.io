Gridea Markdown 语法简单教程

分享
2019-09-23

Markdown 语法参考地址
简书 Markdown创建表格

1. 斜体和粗体

使用 * 和 **表示斜体和粗体。

示例：

这是 斜体，这是 粗体。
*这是斜体的文字*
**这是加粗的文字**

2. 标题

使用 #加空格表示一级标题，同理，##加空格表示二级标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

3. 外链接

使用[描述](链接地址）为文字增加外链接。

示例：

这是去往 Gridea作者主页 的链接。

4. 无序列表

使用 星号*/加号+/减号- 表示无序列表。

星号/加号/减号 + 空格 + 列表内容

示例：

  • 无序列表项 一
  • 无序列表项 二
  • 无序列表项 三

5. 有序列表

使用数字和点表示有序列表。

数字 + 英文句号 + 空格 + 列表内容

示例：

  1. 有序列表项 一
  2. 有序列表项 二
  3. 有序列表项 三

6. 文字引用

使用 > 表示文字引用。

示例：

野火烧不尽，春风吹又生。

7. 行内代码块

使用 `代码` 表示行内代码块。

示例：

让我们聊聊 html。

8. 代码块

使用 四个缩进空格 或 ``` 表示代码块。
```这是一个代码块，此行左侧有四个不可见的空格。```
示例：

这是一个代码块，此行左侧有四个不可见的空格。

测试代码

function hello() {
	var username = 'xiaoming';
	var age = 18;
	console.log('Hello World');
}
hello()

11. 嵌套列表

标记语法：

#### 嵌套列表
+ 我是第一层列表
	+ 我是第二层列表
		+ 我是第三层列表
			+ 我是第四层列表

嵌套列表示例

  • 我是第一层列表
    • 我是第二层列表
      • 我是第三层列表
        • 我是第四层列表
#### 有序列表和无需列表相互嵌套
1. 我是第一层有序列表
	+ 我是第二层无序列表
		1. 我是第三层有序列表
		2. 我也是第三层有序列表

12. 补充说明

在使用markdown书写博客时，有可能会出现文本中的字符是markdown语法，那么markdown翻译器就会误将这些符号也翻译成某种功能，就会出现显示错误的情况。此时需要使用反斜杠（''）来将这些字符进行转义。可以使用反斜杠（''）转义的符号如下：

\   反斜线
`   反引号
*   星号
_   下划线
{}  大括号
[]  中括号
()  小括号
#   井号
+   加号
-   减号（连字符）
.   句点
!   感叹号
